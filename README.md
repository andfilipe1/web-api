

# Exercicio de criação de uma WebApi utilizando aspnet core e um banco em memoria.


GET /api/usuarios

Obtem todos os usuarios

GET /api/usuarios/{id}

Obter um usuario pelo ID

POST /api/usuarios

Adiciona um novo usuário

PUT /api/usuarios/{id}

Atualiza um usuario

DELETE /api/usuarios/{id}

Exclui um usuário
